Данные для проекта использовались из соревнования ods.ai/competitions/x5-retailhero-uplift-modeling

Скачать данные можно по ссылке https://disk.yandex.ru/d/ivrJ2q3k9YoUjw
Для воспроизведения результатов скачанные данные, необходимо расположить в папке /data/raw/

Структура проекта:

/data/ - данные (сырые и подготовленные)

/models/pkl - сохраненное состояние обученных моделей

/notebooks - модули сервиса (обработка данных, обучение моделей, деплой)

/src - скрипты, исходники
